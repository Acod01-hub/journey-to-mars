# Journey to Mars – An Interactive Web Odyssey
An immersive, interactive web experience exploring humanity's journey to the Red Planet. Built with React, Vite, Tailwind CSS, and Framer Motion.

✨ Features
- **Custom Loading Animation** - Animated rocket launch sequence
- **Parallax Star Field** - Multi-layered scrolling background with twinkling stars and nebulas
- **Scroll-Triggered Animations** - Smooth entrance animations for all sections
- **Interactive Mission Cards** - Hover effects and expandable content
- **Spacecraft Details Toggle** - Clickable sections showing propulsion, life support, navigation, and habitat info
- **Mars Facts Carousel** - Auto-advancing slider with navigation controls
- **SVG Orbit Animations** - Animated orbital paths with satellites
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices

## 🛠️ Tech Stack
- **React 18** - UI library
- **Vite 5** - Build tool and dev server
- **Tailwind CSS 3** - Utility-first styling
- **Framer Motion 11** - Animation library
